# 📱 Konter Tycoon v2

Selamat datang di **Konter Tycoon**, sebuah game simulasi bisnis jual-beli handphone seken langsung dari browser kamu! Game ini dirancang ringan menggunakan kombinasi HTML, CSS, dan JavaScript murni.

## 🎮 Cara Bermain
1. Kamu akan memulai bisnis dengan modal awal **Rp 5.000.000**.
2. **Beli HP Bekas:** Pelanggan akan datang menawarkan HP mereka. Perhatikan kondisi fisik dan harganya. Jika murah, beli untuk dijadikan stok toko.
3. **Fitur Nego:** Merasa harga dari pelanggan kurang cocok? Gunakan tombol **Nego Harga** untuk mendapatkan keuntungan lebih besar. Tapi hati-hati, jika terlalu banyak nego, pelanggan bisa kesal dan kabur!
4. **Sistem Servis & Perbaikan:** Jika kamu memiliki stok HP rusak (Layar Retak / Mati Total), kamu bisa memperbaikinya terlebih dahulu dengan biaya **Rp 400.000** saat ada calon pembeli datang. HP yang sudah diservis harganya akan melonjak tajam!
5. **Jual & Cari Cuan:** Jual koleksi HP di etalase kamu kepada pembeli yang datang dengan tawaran tertinggi. Jangan sampai modalmu habis dan toko bangkrut!

## 🛠️ Teknologi yang Digunakan
* HTML5
* CSS3 (Responsive Design)
* JavaScript Murni (Vanilla JS)
